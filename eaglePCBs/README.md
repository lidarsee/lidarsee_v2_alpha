# lidarsee_v2_hardware

Motherboard_STM32F405RG_r1 ist the mainboard for the head unit. 
### Features
* XV-11 LIDAR compartible
* STM32F405 Microprocessor
* ESP32 WIFI/Bluetooth-Module
* Single cell lipo power supply
* MPU6050 IMU
